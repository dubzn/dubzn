# Santiago Galvan

**Backend Engineer | Blockchain**

I'm a backend engineer with 5 years of experience building and operating
production APIs, background workers, payment flows, release pipelines, and
blockchain infrastructure.

At [Caravana Studio](https://caravana.studio), I work as the founding
backend engineer on [Jokers of Neon](https://jokersofneon.com/), a production
onchain card game available on the App Store, Google Play, and desktop that has
processed 250K+ Starknet mainnet transactions since December 2025.

For my next role, I'm interested in backend and blockchain infrastructure
positions that combine production ownership, smart contracts, and product
engineering.

## Production Engineering

My work on Jokers of Neon spans the backend and onchain infrastructure behind
the game:

- TypeScript/Node.js APIs, background workers, transaction queues, scheduled
  jobs, payment flows, and third-party integrations.
- Communication between frontend clients, PostgreSQL/Supabase, Torii indexers,
  and Cairo smart contracts.
- Dedicated Katana and Torii deployments on AWS EC2 with Nginx, TLS, rate
  limiting, encrypted EBS volumes, and restricted Security Groups.
- Performance-critical Cairo flows, including onchain caching that reduced
  processing time and gas usage by approximately 50%.
